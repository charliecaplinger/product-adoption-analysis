# Product Adoption Analysis (SaaS Analytics)

This project analyzes user engagement and feature adoption for a simulated SaaS product using SQL, Python, and Tableau.

## Dataset

The dataset includes:

- User signup dates
- Feature usage events
- Subscription plans (Free, Pro, Enterprise)
- Geographic regions

## Key Questions

- Which features are most widely adopted?
- How does usage differ across subscription plans?
- What does daily user engagement look like?
- How many users return after signup?

## Analysis

- Calculated feature adoption rates using SQL
- Measured Daily Active Users (DAU)
- Segmented feature usage by plan and region
- Analyzed retention using time-based metrics

## Key Insights

- Search is the most widely adopted feature across all plans
- Enterprise users show higher engagement across multiple features
- Free users tend to use fewer features
- Daily engagement trends highlight consistent user activity

## Skills Demonstrated

- Product analytics  
- SQL aggregation and segmentation  
- Python (Pandas) for data analysis  
- Data visualization and dashboarding
