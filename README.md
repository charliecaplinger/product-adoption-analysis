# Product Adoption Analysis

This project analyzes user engagement data to understand feature adoption and user behavior trends.

## Tools Used
SQL  
Python (Pandas)  
Tableau  

## Analysis

- Calculated feature adoption rates using SQL
- Analyzed user engagement trends using Python
- Measured daily active users and feature usage

## Key Insights

- Search is the most widely used feature
- Some features show lower adoption, indicating opportunities for improvement
- User engagement varies across days

## Skills Demonstrated

- Data analysis  
- SQL aggregation  
- Data visualization  
- Exploratory Data Analysis (EDA)
